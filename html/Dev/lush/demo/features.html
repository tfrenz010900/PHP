Perfomance

Lush slider was designed to use native transitions to brings the best perfomance in animations vs. the classic jQuery animations.

Responsive

From small to widescreen, Lush slider will adapt itself and all its components to the best size in all screen resolution. Build your slider once and use it everywhere.

Syntax

Forget about complex configurations. Lush syntax is semantics, which means that you can describe animation events in the same way you would think it.

3D Animations

Take advantage of the latest CSS3 improvements in terms of animation and slide in 3 directions!

Extensible

Immerse yourself in the world of animation. Leverages the extension system to expands the animations set without limits and make your slider unique!

Maintainable

Via the extension system you can easily add new animations, enhance or drop what you don't use. So simple like add and remove CSS classes.

Flexslider

You may noticed that Flexslider captions are encouraged along with the images, but you can't encourage them separately. Well, Now you can reusing your markup.

Fallbacks

What happens in older browsers? Don't worry about it, Lush will use fallbacks animations, more simple but attractive to make your slider works anyway.

Skin

Lush skin system is based on CSS, you can use images as is by default, or just adding or changing colors, shadow, etc, you got a new skin. All skin images include its psd version so you can edit them.

Flexible

You can control the slider from the standar navigations controls or add set new components to be used as controls, like buttons, images, links, etc. You decide what controls Lush!

API

Lush API gives you full control over the slider, pause, resume, stop, advance, etc., also you can take actions of what happens inside the slider with callback and events.

Documention

But, how will I use all this functions? Just got the docs, it's extensive and precise with examples to guide you building your own Lush Slider. And if you don't find what you look, you always can ask for support.
